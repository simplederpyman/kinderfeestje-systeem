# 🎮 Spelstructuur — Volledige Flow

## Overzicht

Het spel bestaat uit **10 stappen**. Elke stap combineert een fysieke printable met invoer op de website. De website bevestigt elke juiste code en geeft instructies voor de volgende stap.

---

## Stap 1 — De Startbrief

| | |
|---|---|
| **Wat kinderen doen** | Openen de eerste envelop die op tafel ligt |
| **Printable** | Envelop 1 — Startmissie |
| **Wat ze ontdekken** | Een welkomstbrief van Professor Kwarts met uitleg over het spel |
| **Code** | `KWARTS-01` (staat in de brief) |
| **Website invoer** | Voeren `KWARTS-01` in op de startpagina |
| **Website feedback** | "Welkom, speurneuzen! Professor Kwarts heeft jullie verwacht. Ga naar de keuken — daar ligt iets bijzonders onder de rode theedoek. Zoek Envelop 2." |

---

## Stap 2 — Het Alfabetwiel

| | |
|---|---|
| **Wat kinderen doen** | Vinden Envelop 2 in de keuken, openen hem |
| **Printable** | Envelop 2 — Het Alfabetwiel + Cijfercode |
| **Wat ze ontdekken** | Een alfabetwiel (A=1, B=2...) en een reeks cijfers: `8-5-12-16` |
| **Oplossing** | Cijfers omzetten naar letters: H-E-L-P |
| **Code** | `HELP` |
| **Website invoer** | Voeren `HELP` in |
| **Website feedback** | "Professor Kwarts vroeg om hulp! Maar ze heeft meer achtergelaten. Zoek Envelop 3 in de woonkamer, achter het grootste kussen." |

---

## Stap 3 — De Spiegelboodschap

| | |
|---|---|
| **Wat kinderen doen** | Vinden Envelop 3, een brief die in spiegelschrift is geschreven |
| **Printable** | Envelop 3 — Spiegelboodschap |
| **Wat ze ontdekken** | Ze moeten de tekst in een spiegel lezen (of achterstevoren houden via de website) |
| **Oplossing** | De spiegeltekst luidt: "DE SLEUTEL ZIT ONDER DE TRAP" |
| **Code** | `SLEUTEL` |
| **Website invoer** | Voeren `SLEUTEL` in |
| **Website feedback** | "Goed gezien! Onder de trap ligt Envelop 4. Maar pas op — niet alles is wat het lijkt..." |

---

## Stap 4 — Het Geheime Alfabet

| | |
|---|---|
| **Wat kinderen doen** | Vinden Envelop 4 onder de trap met een substitutie-code |
| **Printable** | Envelop 4 — Geheime Alfabetkaart |
| **Wat ze ontdekken** | Een kaart waar elk letter door een symbool is vervangen. Een bericht in symbolen. |
| **Oplossing** | Het symbolenbericht ontcijferd: "DRIE STAPPEN LINKS VAN DE BOOM" |
| **Code** | `BOOM` |
| **Website invoer** | Voeren `BOOM` in |
| **Website feedback** | "Professor Kwarts hield van de natuur. Ga naar de tuin (of het balkon) — Envelop 5 hangt aan iets groens." |

---

## Stap 5 — De Verborgen Plaatjes

| | |
|---|---|
| **Wat kinderen doen** | Vinden Envelop 5 met een afbeelding-puzzel |
| **Printable** | Envelop 5 — Plaatjescode (6 afbeeldingen die een zin vormen) |
| **Wat ze ontdekken** | 6 plaatjes die samen een zin vormen: "Volg de sterren naar de zolder" |
| **Code** | `STERREN` |
| **Website invoer** | Voeren `STERREN` in |
| **Website feedback** | "De sterren wijzen de weg! Op zolder wacht Envelop 6. Maar de deur is op slot — het cijferslot is 7-3-9-1." |

---

## Stap 6 — Het Cijferslot

| | |
|---|---|
| **Wat kinderen doen** | Gaan naar zolder (of een afgesloten ruimte), gebruiken het cijfer om een slotje te openen |
| **Printable** | Envelop 6 — Zit achter het slotje |
| **Wat ze ontdekken** | Een brief met een raadsel: "Ik heb tanden maar bijt niet. Ik heb een mond maar spreek niet. Wat ben ik?" |
| **Oplossing** | Een rits (of: een kam — kies wat past in je huis) |
| **Code** | `RITS` (of `KAM`) |
| **Website invoer** | Voeren de oplossing in |
| **Website feedback** | "Slim! Professor Kwarts was dol op raadsels. Zoek Envelop 7 — die zit in de badkamer, in een potje." |

---

## Stap 7 — De Sleutelzoektocht

| | |
|---|---|
| **Wat kinderen doen** | Vinden Envelop 7 in de badkamer met een plattegrond |
| **Printable** | Envelop 7 — Miniplattegrond van het huis (vereenvoudigd) |
| **Wat ze ontdekken** | Op de plattegrond zijn X-markeringen. Ze moeten de echte locaties vinden (3 sleutels verstoppen). |
| **Code** | De drie sleutels samen vormen: `DEUR` |
| **Website invoer** | Voeren `DEUR` in |
| **Website feedback** | "Jullie zijn echte detectives! Achter de voordeur (of in de gang) ligt Envelop 8." |

---

## Stap 8 — De Tijdcapsule

| | |
|---|---|
| **Wat kinderen doen** | Vinden Envelop 8 met een "tijdcapsule" — een brief "uit het verleden" |
| **Printable** | Envelop 8 — Brief van Professor Kwarts (6 maanden geleden geschreven) |
| **Wat ze ontdekken** | De brief vertelt over een experiment dat mis ging. Er staat een rebus in verstopt. |
| **Oplossing** | Rebus oplossing: "KRISTAL" |
| **Code** | `KRISTAL` |
| **Website invoer** | Voeren `KRISTAL` in |
| **Website feedback** | "Het kristal! Professor Kwarts sprak altijd over de Kwartssteen. Nog 2 stappen. Zoek Envelop 9 — die is verstopt waar je tv kijkt." |

---

## Stap 9 — Het Laatste Raadsel

| | |
|---|---|
| **Wat kinderen doen** | Vinden Envelop 9 bij de tv-hoek |
| **Printable** | Envelop 9 — Groot kruiswoordraadsel |
| **Wat ze ontdekken** | Een kruiswoordraadsel. De gemarkeerde letters vormen een zin. |
| **Oplossing** | De zin: "DE SCHAT IS WAAR HET BEGON" |
| **Code** | `BEGIN` |
| **Website invoer** | Voeren `BEGIN` in |
| **Website feedback** | "Waar het begon... Jullie eerste envelop! Ga terug naar de tafel. Daar ligt de laatste envelop — Envelop 10. Jullie zijn er bijna!" |

---

## Stap 10 — De Kwartssteen

| | |
|---|---|
| **Wat kinderen doen** | Gaan terug naar de starttafel, vinden Envelop 10 |
| **Printable** | Envelop 10 — Slotbrief + aanwijzing naar de schat |
| **Wat ze ontdekken** | De brief zegt: "Onder de tafel ligt een geheim luikje" (een doos die onder de tafel is vastgeplakt) |
| **Code** | `KWARTS` |
| **Website invoer** | Voeren `KWARTS` in |
| **Website feedback** | 🎉 **OVERWINNING!** "JULLIE HEBBEN HET GEVONDEN! De Kwartssteen is van jullie. Professor Kwarts is trots op jullie!" — Confetti op het scherm + eindceremonie |

---

## Samenvatting Codes

| Stap | Code | Locatie envelop |
|------|------|-----------------|
| 1 | `KWARTS-01` | Tafel (start) |
| 2 | `HELP` | Keuken |
| 3 | `SLEUTEL` | Woonkamer (kussen) |
| 4 | `BOOM` | Onder trap |
| 5 | `STERREN` | Tuin/balkon |
| 6 | `RITS` | Zolder (slotje) |
| 7 | `DEUR` | Badkamer |
| 8 | `KRISTAL` | Gang/vooruit |
| 9 | `BEGIN` | TV-hoek |
| 10 | `KWARTS` | Terug op tafel |
