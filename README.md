# 🔮 De Geheime Code van Professor Kwarts

## Hybride Kinderfeestje Systeem

Een volledig hybride kinderfeestje waarbij een interactieve website samenwerkt met fysieke printables, enveloppen en puzzels.

### Productnaam
**De Geheime Code van Professor Kwarts**

### Thema
Mysterie & Avontuur — Geheime codes, verborgen berichten en een schatzoektocht

### Doelgroep
- **Leeftijd:** 8–10 jaar
- **Aantal spelers:** 6–8 kinderen
- **Speelduur:** 2 uur
- **Moeilijkheid:** Gemiddeld

### Missie
Professor Kwarts is spoorloos verdwenen! Ze heeft een reeks gecodeerde berichten achtergelaten voordat ze verdween. Alleen door haar geheime codes te kraken, kunnen de kinderen ontdekken waar ze is — en de verborgen schat vinden die ze voor hen heeft achtergelaten.

### Structuur
- 🌐 **Website** — Centrale hub die het spel aanstuurt
- 🖨️ **Printables** — Enveloppen, puzzels, missiekaarten
- 👨‍👩‍👧 **Ouderhandleiding** — Stap-voor-stap begeleiding
- 🧩 **10 puzzels** — Logisch oplopend in moeilijkheid

### Mappenstructuur
```
kinderfeestje-systeem/
├── docs/                    # Verhaallijn, spelstructuur, puzzels
├── printables/              # Alle printable teksten en beschrijvingen
├── website/                 # Website structuur en teksten
├── ouderhandleiding/        # Oudergids
└── README.md               # Dit bestand
```

### Snel starten
1. Lees `docs/01-verhaallijn.md` voor het verhaal
2. Bekijk `docs/02-spelstructuur.md` voor de volledige flow
3. Print alles uit `printables/`
4. Zet de website in `website/` online (of open lokaal)
5. Volg `ouderhandleiding/01-voorbereiding.md`
